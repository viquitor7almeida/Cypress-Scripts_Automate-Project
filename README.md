# Cypress Scripts for Automate Application

Este repositório contém os scripts de automação de testes desenvolvidos com **Cypress** para a aplicação **Automate**, uma plataforma interna da **Hortifruti Natural da Terra**.

---

## Sobre o projeto

O **Automate** é uma aplicação web criada para otimizar o processo de **gestão de entregas e solicitações de produtos** em mais de **70 lojas** da Hortifruti Natural da Terra, distribuídas por **três estados do Brasil**.

A plataforma utiliza **inteligência artificial (IA)** para gerar **sugestões de pedidos automatizadas**, baseadas em **regras de negócio** e **históricos de vendas e estoque**.

---

## Papel do QA

Atuei como **Quality Assurance (QA)** no projeto, sendo responsável por:

- Executar **testes exploratórios** nos diferentes módulos da aplicação.  
- Desenvolver **scripts de automação** utilizando **Cypress**.  
- Criar e manter **cenários de teste automatizados** que validam fluxos críticos do sistema.  
- Contribuir para a **melhoria contínua** do processo de qualidade e integração dos testes na pipeline de CI/CD.
